Vocab

Compound Boolean Expression

Compound boolean expressions can include && (and) or || (or) to execute code

  • &&: code will execute if and only if both conditions are true
  • ||: code will execute if either or both conditions are true
int x = 4;
int y = 9;

if (x >= 0 && y <= 13) {
    System.out.println("Both conditions are true");
}

if (x > 5 || y > 5) {
    System.out.println("At least one of the conditions are true");
}

Both conditions are true
At least one of the conditions are true

Truth Tables

A truth table is a breakdown of a logic function by listing all possible values the function can attain. Such a table typically contains several rows and columns, with the top row representing the logical variables and combinations, in increasing complexity leading up to the final function.
